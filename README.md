Short-Term Forecast of Gaseous Air Pollutants (O₃ and NO₂)

Project Overview

Air pollution is a major environmental issue in urban areas. This project aims to forecast short-term concentrations of gaseous air pollutants, specifically ground-level ozone (O₃) and nitrogen dioxide (NO₂).

The study focuses on Delhi, and aims to predict pollutant levels for the next 24 hours using historical pollution data and meteorological information.


Objectives
	•	Collect historical air pollution data for Delhi
	•	Analyze trends in O₃ and NO₂ levels
	•	Build a prediction model for short-term forecasting
	•	Visualize pollution trends and predicted values


Technologies Used
	•	Python
	•	Pandas
	•	Matplotlib
	•	Scikit-learn
	•	Jupyter Notebook
	•	Git & GitHub
